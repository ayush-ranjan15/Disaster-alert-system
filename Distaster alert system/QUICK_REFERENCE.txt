TROUBLESHOOTING
===============

Issue: "Cannot find symbol" compilation error
Solution: Ensure all package imports are correct
         Check file paths and class names

Issue: ClassNotFoundException at runtime
Solution: Verify bin directory has compiled classes
         Check classpath: -cp bin

Issue: FileNotFoundException for alert_log.txt
Solution: Application creates file automatically
         Check write permissions in directory

Issue: Exception in thread "main"
Solution: Check stack trace for specific error
         Ensure all dependencies are available
